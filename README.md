# ml-Healthcare-Premium-Prediction
Machine Learning project for Healthcare Premium Prediction based on factors like age, smoking habits, BMI, and medical history.

### URL for prediction model
https://jalal-ml-healthcare-premium-prediction.streamlit.app/

### Prediction Model Parameters
The prediction of healthcare premium is based on the following parameters:
Age
Number of Dependants
Income
Genetical Rish
Insurance Plan
Employment Status
Gender
Marital Status
BMI Category
Smoking Status
Region and
Medical History
